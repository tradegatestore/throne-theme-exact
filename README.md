# Throne Theme - Luxury Dark Shopify Theme

A premium dark luxury Shopify theme inspired by the King Throne preset, featuring elegant gold accents, modern animations, and a sophisticated e-commerce experience.

## Features

- **Dark Luxury Aesthetic** - Deep black backgrounds with elegant gold accents
- **Modern Animations** - Smooth scroll reveals, hover effects, and transitions
- **Fully Responsive** - Optimized for all devices from mobile to desktop
- **Section-Based Design** - Modular sections for easy customization
- **Performance Optimized** - Fast loading with lazy loading images
- **SEO Friendly** - Proper semantic HTML and meta tags

## Sections Included

- Header with mega menu support
- Announcement bar
- Slideshow / Hero banner
- Video hero section
- Featured collection
- Collection list
- Image with text
- Rich text
- Testimonials
- Newsletter signup
- Marquee text
- Dual media slideshow
- Footer with multiple columns

## Installation

### Method 1: Download ZIP
1. Click the green **Code** button above
2. Select **Download ZIP**
3. Go to your Shopify Admin
4. Navigate to **Online Store** > **Themes**
5. Click **Add theme** > **Upload ZIP file**
6. Select the downloaded ZIP file
7. Click **Customize** to start editing

### Method 2: GitHub Integration
1. In Shopify Admin, go to **Online Store** > **Themes**
2. Click **Add theme** > **Connect from GitHub**
3. Authorize Shopify to access your GitHub
4. Select this repository

## Theme Structure

```
throne-theme-exact/
|-- assets/           # CSS, JS, and image files
|-- config/           # Theme settings schema
|-- layout/           # Theme layout templates
|-- locales/          # Translation files
|-- sections/         # Modular section files
|-- snippets/         # Reusable code snippets
|-- templates/        # Page templates (JSON)
```

## Customization

After installing, you can customize the theme through:

1. **Theme Editor** - Go to Online Store > Themes > Customize
2. **Theme Settings** - Access colors, typography, and global settings
3. **Section Settings** - Each section has individual customization options

## Color Scheme

- Primary Background: `#0d0d0d` (Deep Black)
- Secondary Background: `#1a1a1a` (Dark Gray)
- Accent Color: `#c9a961` (Gold)
- Text Color: `#ffffff` (White)
- Secondary Text: `#a0a0a0` (Gray)
